# 🚀 PRIMER PROGRAMA

Todo lenguaje de programación empieza con algo simple:

## 📌 Hello, World

```python
print("Hello, world")
```

---

# 🧠 ¿Qué hace esto?

- `print()` → muestra información en pantalla
- `"Hello, world"` → texto (string)

📌 Resultado:

```
Hello, world
```

---

# 💡 POR QUÉ ES IMPORTANTE

Este pequeño programa sirve para:

- comprobar que Python funciona
- entender la función `print`
- familiarizarte con la sintaxis
